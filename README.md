# Wonders of India

A captivating travel and tourism website showcasing the most beautiful destinations across India. Built with modern web technologies and a focus on smooth user experience.

## Features

- **Responsive Design**: Looks great on desktops, tablets, and mobile devices.
- **Smooth Animations**: Utilizes Locomotive Scroll for a premium, scroll-triggered animation feel.
- **Interactive UI**: Hover effects, smooth transitions, and a clean, modern interface.
- **Three Main Sections**:
  - **Places**: Explore popular Indian tourist spots like Leh-Ladakh, Manali, and more.
  - **Services**: View our premium travel services including hotel bookings and guided tours.
  - **Contact**: Easy-to-use contact form to get in touch with us.

## Tech Stack

- **HTML5** - Semantic markup.
- **CSS3** - Custom properties, transitions, and responsive design.
- **JavaScript (Vanilla)** - Core logic, animations, and DOM manipulation.
- **Locomotive Scroll** - Smooth scrolling and scroll-based animations.

## Getting Started

1.  Clone the repository.
2.  Open `index.html` in your web browser.
3.  Navigate through the different sections and enjoy the experience!
